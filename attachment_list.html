Види сайтів:

  1. Статичні - це сайти, що не мають back-end, тобто не мають бази з даними. Вони створюється через такі програми як:
    • HTML
    • CSS
    • інколи JS...
  2. Динамічні - це сайти, що мають back-end, тобто беруть дані з бази. Вони створюється через:
    • CSS
    • обов'язково JS
    • php чи ruby та інше
